# Static Resume - Quick Reference

Navigate to this project when working on resume updates or deployment.

## Quick Links
- [Project Overview](.claude/project.md)

## Common Commands

```bash
# Start development server
npm start

# Test endpoints
curl http://localhost:3001/md
curl http://localhost:3001/json

# Deploy (push to trigger GitHub Actions)
git push origin main
```

## Resume Variants
1. General Purpose → `resume.tex`
2. App Developer → `resume-app.tex`
3. Backend Engineer → `resume-backend.tex`
4. DevOps Engineer → `resume-devops.tex`
5. Web Developer → `resume-web.tex`
